微信对公收款

PC 扫码支付需要开通 Native 支付功能才能正常使用 。

微信的对公收款要求必须有已经认证的服务号，同时服务号必须已经认证并开通微信支付功能。 更多请参考 https://pay.weixin.qq.com 。

支付宝对公收款

需要在支付宝申请 【当面付收单】才能开启此功能。

支付宝的合作者ID 和 应用ID 等信息请去 https://open.alipay.com 申请。

注意：不是自己创建的公钥，需要通过支付宝商户后台获取。

请填写支付宝的商户收款账号，一般情况下和 合作者ID 的值相同。

PayPal对公收款

Paypal 的商户号及密钥等请去 https://developer.paypal.com/ 申请。
